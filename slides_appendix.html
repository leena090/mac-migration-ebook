부록. Safari 완전 가이드

Safari — 크롬만 쓰던 당신이 꼭 써봐야 할 이유

"크롬이면 충분한데 왜 사파리?" — 맥에서는 사파리가 압도적으로 유리한 상황이 있습니다.

🔋

배터리 2~3시간 더

Apple Silicon 최적화로 크롬 대비 배터리 소모 30~40% 적음. 카페에서 충전기 없이 하루 작업 가능

🚀

메모리 절약 왕

크롬은 탭 하나에 200~500MB, 사파리는 50~150MB. 탭 30개 열어도 맥이 느려지지 않음

🔒

프라이버시 기본 내장

지능형 추적 방지(ITP)가 광고 추적을 자동 차단. 확장 프로그램 없이도 깔끔

📱

Apple 생태계 연동

iPhone에서 보던 페이지를 맥에서 바로 이어보기. AirDrop으로 링크 즉시 전송

결론: 둘 다 쓰세요!

사파리 = 일반 브라우징, 영상 시청, 리서치, 장시간 작업
크롬 = 구글 서비스(YouTube Studio, Analytics), 확장 필요 시, 웹 개발
기본은 사파리, 특정 작업만 크롬 — 이게 맥 고수들의 패턴입니다.

부록. Safari 워크플로우

Safari 크리에이터 워크플로우

유튜버 · AI 강사 · 전자책 제작자가 사파리를 200% 활용하는 방법

기능 활용법 크리에이터 시나리오
탭 그룹 주제별 탭 묶음 저장 · 전환 "영상 리서치" 그룹, "경쟁 채널" 그룹, "AI 도구" 그룹으로 분리. 작업 전환 시 한 클릭
리더 모드 광고/메뉴 제거 → 텍스트만 표시 블로그 리서치 시 핵심 내용만 빠르게 스캔. 전자책 원고 참고자료 읽기에 최적
웹 앱 (Dock 추가) 사이트를 독립 앱처럼 사용 ChatGPT, Claude, Canva, Notion을 Dock에 앱으로 추가 → 브라우저 탭 낭비 제로
프로필 분리 개인용 / 업무용 프로필 전환 개인 구글 계정 vs 채널 관리 계정을 프로필로 분리. 로그인 충돌 없음
하이라이트 웹페이지 텍스트 하이라이트 공유 참고 자료에서 핵심 문장을 하이라이트 → 팀원에게 정확한 부분 공유
번역 내장 페이지 자체 번역 (확장 불필요) 해외 AI 뉴스, 영어 문서를 바로 번역. 해외 트렌드 리서치 속도 2배

꿀팁: Cmd+Shift+\ → 탭 전체 미리보기. 탭이 20개 넘어도 원하는 페이지를 눈으로 찾아 바로 이동!

부록. Safari 숨은 기능

Safari에만 있는 숨은 기능 8가지

크롬에는 없는, 사파리만의 특별한 기능들입니다.

  1. Handoff (핸드오프) — iPhone에서 보던 웹페이지를 맥에서 바로 이어보기. Dock 왼쪽에 Safari 아이콘이 뜹니다
  2. AirPlay로 영상 미러링 — 사파리에서 재생 중인 영상을 TV/모니터로 바로 전송. 발표/시연에 유용
  3. 자동 강력 암호 — 회원가입 시 초강력 비밀번호 자동 생성 + iCloud 키체인에 저장. 모든 Apple 기기에서 자동 입력
  4. 패스키 (Passkey) — 비밀번호 없이 Touch ID/Face ID로 로그인. 피싱 100% 차단
  5. 에너지 소모 표시 — 탭별 에너지 사용량 확인 가능. 배터리 잡아먹는 사이트를 바로 찾아 닫기
  6. 웹 앱 알림 — Dock에 추가한 웹 앱에서 macOS 알림 수신 가능 (크롬은 브라우저 켜야 함)
  7. 비디오 PIP 자동 — 영상 재생 중 다른 탭으로 이동하면 자동으로 PIP(화면 속 화면) 전환. 튜토리얼 보면서 따라하기에 최고
  8. 프라이빗 브라우징 잠금 — 사파리 비공개 탭을 Touch ID로 잠금. 공유 맥에서도 안심

부록. 크리에이터 맥 활용

유튜버가 맥북을 써야 하는 진짜 이유

유튜브 운영 · 영상 편집 · 썸네일 제작 — 맥북이 윈도우보다 유리한 이유가 있습니다.

🎬

Final Cut Pro = 맥 전용

프리미어보다 빠른 렌더링, 마그네틱 타임라인, Apple Silicon 완벽 최적화. M5 Pro에서 4K 60fps 실시간 편집

🎨

레티나 디스플레이

색 정확도 P3 와이드 컬러. 썸네일 색상이 모니터마다 다르게 보이는 문제 최소화. 캘리브레이션 불필요

🔇

무소음 편집

M5 Pro는 일반 편집 시 팬 소리 제로. 마이크 녹음 중에도 노트북 소음이 들어가지 않음

🔋

어디서든 편집

22시간 배터리 → 카페, 촬영 현장에서 충전기 없이 편집 가능. 윈도우 노트북은 편집 시 2~3시간이 한계

📱

iPhone 연동

AirDrop으로 촬영 영상을 초고속 전송. iPhone으로 찍고 → 맥에서 바로 편집. 케이블/USB 불필요

🎙️

내장 마이크 품질

맥북프로 내장 마이크 = 저가 콘덴서급. 급할 때 외장 마이크 없이도 나레이션 녹음 가능

핵심: 맥북은 "영상 편집 전용 기기"가 아닙니다. 편집 + 촬영 연동 + 이동성 + 무소음이 결합된 크리에이터 올인원 워크스테이션입니다.

부록. AI 강사 맥 활용

AI 강사 & 전자책 제작자 — 맥이 유리한 7가지

AI 도구 활용, 강의 제작, 전자책 집필 — 맥에서 더 효율적인 이유

# 활용 영역 맥북의 장점
1 Claude Code 터미널 맥 터미널(zsh)은 리눅스 기반 → Claude Code와 AI 개발 도구가 설치 한 줄이면 끝. 윈도우는 WSL 설정부터 복잡
2 화면 녹화 강의 Cmd+Shift+5로 별도 프로그램 없이 고화질 화면 녹화. OBS 없이도 강의 영상 제작 가능
3 전자책 PDF 제작 맥의 Pages/Keynote → PDF 내보내기 품질이 윈도우 대비 우수. 인쇄용 CMYK 프로파일 기본 지원
4 Keynote 프레젠테이션 파워포인트보다 세련된 템플릿 + 애니메이션. 온라인 강의 슬라이드 제작에 최적. 무료!
5 Continuity Camera iPhone을 웹캠으로 사용! 줌/구글미트 강의 시 맥북 웹캠 대신 iPhone 카메라 연결 → 화질 10배 향상
6 유니버설 클립보드 iPhone에서 복사 → 맥에서 붙여넣기. 스크린샷/텍스트를 기기 간 자유롭게 이동. 강의 자료 준비 속도 UP
7 Homebrew 생태계 brew install 한 줄로 ffmpeg, node, python, pandoc 설치. AI 파이프라인 구축이 윈도우보다 3배 빠름

Continuity Camera 꿀팁: iPhone을 맥북 위에 거치 → 시스템 설정 > 카메라에서 iPhone 선택. 별도 앱 설치 없이 바로 고화질 웹캠!

부록. 빠진 꿀팁 보충

본편에서 미처 다루지 못한 맥 필수 팁

시스템 & 생산성

  • Sidecar (사이드카) — iPad를 맥의 보조 모니터로 사용. 듀얼 모니터 없이 화면 확장. 영상 편집 시 타임라인을 iPad에 표시
  • Focus Mode (집중 모드) — "영상 편집" 모드 만들기 → 특정 앱 알림만 허용. 편집 중 카톡/메일 알림 자동 차단
  • 빠른 메모 (Quick Note) — 핫코너나 fn+Q로 어디서든 메모 팝업. 리서치 중 아이디어 즉시 기록
  • 음성 제어 (Voice Control) — "다음 슬라이드", "스크롤 다운" 등 음성으로 맥 제어. 손이 바쁠 때 유용
  • Automator / 단축어 — 반복 작업 자동화. 예: "매주 월요일 유튜브 분석 폴더 자동 생성" 같은 루틴 설정

파일 & 미디어

  • 태그 시스템 — 파일/폴더에 색상 태그 지정. "빨강=긴급", "파랑=편집중", "초록=완료" 식으로 프로젝트 관리
  • 스마트 폴더 — 조건에 맞는 파일을 자동 수집. "최근 7일 내 .mp4 파일" 폴더 → 촬영 영상 자동 정리
  • 미리보기(Preview) 편집 — PDF 합치기, 이미지 크기 조정, 서명 삽입이 기본 앱으로 가능. 포토샵/아크로뱃 없이도 OK
  • HEIF/HEVC 기본 지원 — iPhone 촬영 포맷을 변환 없이 바로 편집. 윈도우는 코덱 설치 필요
  • AirDrop 대용량 전송 — 10GB 영상 파일도 AirDrop으로 iPhone↔Mac 초고속 전송. USB 메모리/클라우드 업로드 불필요

부록. 최종 체크리스트

크리에이터 맥 세팅 — 최종 체크리스트

유튜버 · AI 강사 · 전자책 제작자라면, 맥 초기 세팅 시 이것까지 챙기세요.

영상 편집 세팅

  • Final Cut Pro 설치 (App Store, 1회 구매)
  • Compressor 설치 (다양한 내보내기 프리셋)
  • 영상 소스 폴더 구조 통일: ~/Movies/프로젝트명/
  • 외장 SSD 연결 → 라이브러리를 외장으로 이동 (내장 용량 절약)
  • Cmd+Shift+5 녹화 설정: 마이크 선택, 저장 위치 지정

AI 도구 세팅

  • Homebrew 설치 → brew install node python git
  • Claude Code 설치 → npm i -g @anthropic-ai/claude-code
  • ChatGPT, Claude, Gemini → Safari 웹 앱으로 Dock에 추가
  • Oh My Zsh 설치 → 터미널 생산성 극대화

강의 & 전자책 세팅

  • Keynote 기본 설치 확인 (강의 슬라이드)
  • Continuity Camera 설정 (iPhone 웹캠)
  • Focus Mode → "강의 모드" 프로필 생성
  • 화면 녹화 시 "마우스 클릭 표시" 켜기 (시스템 설정 > 접근성)
  • Pages로 전자책 → PDF 내보내기 테스트

Safari 세팅

  • 탭 그룹 3개 생성: "리서치", "경쟁 분석", "도구"
  • 프로필 분리: "개인용" vs "채널 관리용"
  • 자주 쓰는 AI 도구 → Dock에 웹 앱 추가
  • 리더 모드 자동 켜기: 설정 > 웹사이트 > 리더
