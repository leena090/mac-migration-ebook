처음 켜면 바로 하세요! 10가지 설정

맥을 처음 켠 뒤, 아래 10가지만 설정하면 훨씬 편하게 쓸 수 있습니다.

  1. 스크롤 방향 바꾸기
    시스템 설정 → 트랙패드 → 스크롤 및 확대/축소 → "자연스러운 스크롤" 끄기
    (윈도우처럼 아래로 밀면 아래로 갑니다)
  2. 우클릭 활성화
    시스템 설정 → 트랙패드 → 포인트 및 클릭 → 보조 클릭 → "두 손가락으로 클릭"
  3. Dock 크기 및 위치 조절
    시스템 설정 → 데스크톱 및 Dock → 크기 줄이기 + 위치: 왼쪽 추천
    (화면 공간을 더 넓게 활용)
  4. FileVault 켜기 (보안)
    시스템 설정 → 개인정보 보호 및 보안 → FileVault 켜기
    (디스크 전체 암호화 — 분실 시 데이터 보호)
  5. 핫코너 설정
    시스템 설정 → 데스크톱 및 Dock → 핫코너
    (마우스를 화면 모서리에 대면 기능 실행)
  1. 배터리 퍼센트 표시
    시스템 설정 → 제어 센터 → 배터리 → "퍼센트 보기" 켜기
  2. 잠금 시간 설정
    시스템 설정 → 잠금 화면 → 화면 보호기 시작 후 암호 요구 → "즉시"
  3. Night Shift 켜기
    시스템 설정 → 디스플레이 → Night Shift → 예약: 일몰~일출
    (밤에 눈이 편안해집니다)
  4. iCloud 설정
    시스템 설정 → Apple ID → iCloud → 필요한 항목만 켜기
    (사진, 문서 자동 백업)
  5. Time Machine 백업
    시스템 설정 → 일반 → Time Machine → 외장하드 연결 후 백업 디스크 지정
    (실수로 삭제해도 복구 가능!)

Finder vs 탐색기

맥의 Finder는 윈도우의 파일 탐색기와 같은 역할입니다. 하지만 몇 가지 다른 점이 있어요.

항목 Windows 탐색기 Mac Finder
사이드바 즐겨찾기, 내 PC, 네트워크 즐겨찾기, iCloud, 태그 (색상별 정리!)
경로 표시 주소창에 항상 표시 기본 숨김! → 보기 > 경로 막대 켜야 함
잘라내기 Ctrl+X 바로 가능 Cmd+C → Cmd+Option+V
(복사 후 "이동 붙여넣기")
탭 Windows 11부터 지원 Cmd+T로 새 탭 (오래전부터!)
미리보기 미리보기 패널 (느림) Quick Look — Space바 한 번이면 끝!
검색 파일 탐색기 검색 (느림) Spotlight 연동 (초고속!)
숨김 파일 폴더 옵션에서 설정 Cmd+Shift+.

핵심 차이: Finder에는 "잘라내기(Cmd+X)"가 없습니다! 파일 이동은 Cmd+C 후 Cmd+Option+V로 합니다. 다음 슬라이드에서 자세히 설명합니다.

맞습니다. Finder에서는 파일 잘라내기(Cmd+X)가 작동하지 않습니다.
대신 맥만의 독특한 방법이 있어요:

파일 이동 방법

Cmd+C → Cmd+Option+V

복사(Copy) → 이동 붙여넣기(Move)

쉽게 기억하세요

"복사는 그냥 V, 이동은 Option+V"
Option 키를 추가로 누르면 "복사" 대신 "이동"이 됩니다.

Safari vs Chrome — 맥에서 뭘 쓸까?

윈도우에서는 크롬만 썼지만, 맥에는 Safari(사파리)라는 강력한 기본 브라우저가 있습니다.

비교 항목 Safari (사파리) Chrome (크롬)
속도 맥에 최적화되어 체감 속도 빠름 일반적으로 빠르지만 맥에서는 사파리보다 약간 느림
배터리 최대 2~3시간 더 오래감 (Apple 최적화) 배터리 소모가 상대적으로 큼
확장 프로그램 종류가 적음 (App Store 경유) 수만 개 확장 프로그램 지원
동기화 iPhone, iPad와 자동 연동 안드로이드, 윈도우 PC와 연동
프라이버시 추적 차단 기본 내장 구글 광고 기반 → 추적 많음

결론

배터리 + 프라이버시가 중요하면 → Safari  |  확장 프로그램 + 호환성이 중요하면 → Chrome
둘 다 설치해두고 상황에 따라 쓰는 게 가장 좋습니다!

Safari 전용 꿀팁 7가지

사파리에만 있는 편리한 기능들 — 알면 크롬으로 안 돌아갑니다!

  1. 탭 그룹 — 탭이 너무 많을 때, 주제별로 묶어서 정리할 수 있어요. (예: "쇼핑" 그룹, "뉴스" 그룹)
  2. 리더 모드 — 광고와 메뉴를 싹 치우고 글만 깔끔하게 읽을 수 있어요. 주소창 왼쪽 아이콘 클릭!
  3. 2손가락 뒤로/앞으로 — 트랙패드에서 두 손가락으로 좌우 스와이프하면 이전/다음 페이지로 이동
  4. 탭 미리보기 — 열려있는 탭 위에 마우스를 올리면 내용이 미리보기로 보여요
  5. 프라이빗 브라우징 — Cmd + Shift + N → 방문 기록이 남지 않는 비밀 창
  6. 웹사이트를 앱으로 만들기 — 자주 가는 사이트를 Dock(독)에 앱처럼 추가할 수 있어요
  7. iCloud 탭 — iPhone에서 보던 웹페이지를 맥에서 바로 이어서 볼 수 있어요

CH8. Final Cut Pro

실전: 유튜브 10분 영상 편집

처음부터 끝까지, 실제로 유튜브 영상을 편집하는 순서입니다.

  1. 소스 정리 — 촬영 파일을 폴더별로 정리하고 FCP에 임포트 (Cmd+I)
  2. 거친 편집 (Rough Cut) — 쓸 부분만 골라서(I/O) 타임라인에 배치(E). NG, 침묵 구간 과감히 삭제
  3. 정밀 편집 (Fine Cut) — 블레이드(B)로 불필요한 부분 자르고, 트림(T)으로 연결 부분 다듬기
  4. 자막 추가 — Edit → Captions으로 자동 자막 생성, 오타 교정
  5. 배경음악 + 효과음 — 음악 파일을 연결 클립(Q)으로 추가. 볼륨은 -12~-18dB 정도
  6. 색보정 + 인트로/아웃트로 — Inspector에서 색상 보정. Titles에서 인트로·엔딩 카드 추가
  7. 내보내기 — Cmd+E → H.264, 1080p60, 15~20Mbps로 내보내기 → 유튜브 업로드!

소요 시간 기준: 10분 영상 기준 약 2~4시간 (익숙해지면 1~2시간). 캡컷보다 초기 러닝커브가 있지만, 결과물 퀄리티는 확실히 올라갑니다!

윈도우 사용자 실수 TOP 10

맥으로 넘어온 분들이 가장 많이 하는 실수! 미리 알면 당황하지 않아요.

  1. X 버튼 = 종료라고 착각 — 맥에서 X(빨간 버튼)는 창 닫기일 뿐, 앱은 계속 실행 중! 종료는 Cmd+Q
  2. Ctrl 대신 Cmd 써야 함 — 복사 Cmd+C, 붙여넣기 Cmd+V. Ctrl 키는 맥에서 다른 용도!
  3. 오른쪽 클릭이 안 됨 — 트랙패드 두 손가락 클릭 = 오른쪽 클릭. 또는 Ctrl+클릭
  4. 잘라내기(Ctrl+X)가 없음 — 맥은 복사(Cmd+C) 후 Cmd+Option+V = 이동
  5. .exe 파일 실행 시도 — 맥은 .exe를 실행할 수 없어요. 맥용 앱(.dmg/.pkg)을 따로 다운받아야 합니다
  6. NTFS USB가 읽기 전용 — exFAT으로 포맷하면 해결! (CH7 참고)
  7. Delete 키가 Backspace처럼 동작 — 앞 글자 삭제는 Fn+Delete
  8. Alt+F4가 안 됨 — 맥에서 앱 종료는 Cmd+Q
  9. 작업관리자를 못 찾음 — Cmd+Option+Esc = 강제 종료 / 활성 상태 보기 앱도 있어요
  10. 제어판을 못 찾음 — 맥에서는 시스템 설정 (왼쪽 위 Apple 로고 → 시스템 설정)

Safari에만 있는 숨은 기능 8가지

크롬에는 없는, 사파리만의 특별한 기능들입니다.

  7. 비디오 PIP 자동 — 영상 재생 중 다른 탭으로 이동하면 자동으로 PIP(화면 속 화면) 전환. 튜토리얼 보면서 따라하기에 최고
  8. 프라이빗 브라우징 잠금 — 사파리 비공개 탭을 Touch ID로 잠금. 공유 맥에서도 안심

부록. 내장 카메라 & 마이크

맥북 내장 카메라 & 마이크로 영상 촬영하기

맥북프로 M5 Pro에는 12MP 웹캠 + 스튜디오급 3마이크 어레이가 내장되어 있습니다. 별도 장비 없이도 촬영 가능!

내장 카메라 (12MP FaceTime HD)

  • 화질: 1080p (기존 맥 720p에서 대폭 업그레이드). 화상 강의/미팅에 충분
  • Center Stage: 움직이면 자동으로 따라가며 프레임 유지. 강의 중 이동해도 OK
  • Desk View: 책상 위를 내려다보는 뷰 제공. 손으로 작업하는 모습 촬영에 유용
  • Studio Light: 소프트웨어로 얼굴 조명 보정. 역광이나 어두운 곳에서도 얼굴 밝게
  • 한계: 전문 유튜브 영상에는 부족 → Continuity Camera로 iPhone 연결 추천

내장 마이크 (3마이크 어레이)

  • 품질: 지향성 빔포밍 → 내 목소리에 집중, 주변 소음 자동 차단
  • SNR: 신호 대 잡음비가 높아 저가 USB 마이크보다 오히려 깨끗
  • 나레이션 녹음: 조용한 환경이면 내장 마이크만으로 전자책 오디오북, 강의 녹음 가능
  • 음성 격리: macOS 설정에서 "음성 격리" 모드 → 타자 소리, 에어컨 소음 자동 제거
  • 한계: 전문 팟캐스트급은 외장 마이크 필요 → Blue Yeti, Elgato Wave 추천

음성 격리 켜기: 제어 센터 → 마이크 모드 → "음성 격리" 선택. Zoom, FaceTime, 화면 녹화 등 모든 앱에서 자동 적용!
